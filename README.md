# Loan Credit Risk Analysis

## Project Objective
The goal of this project is to analyze loan applicant data and evaluate potential credit risks using data visualization techniques.

## Tools Used
- Power BI
- Data Analysis
- Loan Dataset

## Dataset
Loan dataset containing customer financial information such as income, loan amount, and credit history.

## Dashboard Insights
The dashboard provides analysis on:

- Loan approval status
- Applicant income distribution
- Loan amount trends
- Credit risk indicators

## Results
The analysis helps financial institutions evaluate loan risks and improve credit decision making.

## Applications
- Banking analytics
- Risk management
- Loan approval analysis
